SAMAccountName 通常限制在 20 字符内，UPN 格式为 `username@domain.com`。建议使用小写字母以减少 DNS 不一致问题

| 方案            | 示例                 | 适用场景                         |
| :-------------- | :------------------- | :------------------------------- |
| **拼音全拼**    | `zhangsan`           | 小型公司，员工姓名不重复         |
| **姓. 名**      | `zhang.san`          | 中型公司，可读性好               |
| **名字 + 拼音** | `san.zhang`          | 常见通用形式                     |
| **首字母 + 姓** | `szhang` / `s.zhang` | 姓氏重复较多的环境               |
| **员工编号**    | `10001`              | 对安全性要求极高或无重名校验系统 |

重名处理： 若出现张三（zhangsan）和张三（zhangsan2），可以采用加中间名或员工号的方法，如 `zhangsan01` 或 `san.zhang.2`。
