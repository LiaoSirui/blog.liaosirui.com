## OpenList

聚合多个网盘：将多个网盘（如百度网盘、阿里云盘等）的文件集中管理，方便浏览和搜索 

- Github 仓库：<https://github.com/OpenListTeam/OpenList>
- 文档：<https://doc.oplist.org/>

## 部署

使用 docker 部署：<https://doc.oplist.org/guide/installation/docker#Use%20Docker>

##  使用

添加百度网盘：<https://doc.oplist.org/guide/drivers/baidu#Baidu%20Netdisk>

使用 WebDAV 导出：<https://doc.oplist.org/guide/advanced/webdav>
