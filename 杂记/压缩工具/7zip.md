
## 7z

快速预检（不解压不读全量）

```bash
ls data.7z.* | sort

# 自动检查连续性
ls data.7z.??? | awk -F. '{n=$NF+0; if(n!=++i) print "缺少分卷:", i}'
```

除最后一卷外，所有分卷大小应完全相同（分卷是定长切割的）：

```bash
ls -l data.7z.*
# 或
du -b data.7z.* | sort -k2
```

列出归档内容，比 7z t 快（只读元数据），缺卷时同样会报错，但查不出内容级损坏。

```bash
7z l data.7z.001
```

7z 分卷（.7z.001、.7z.002...）只要对第一卷执行测试，7z 会自动串联所有分卷并校验 CRC：

```bash
7z t data.7z.001
```
