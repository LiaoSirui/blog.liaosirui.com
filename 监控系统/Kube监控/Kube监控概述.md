## 集群层面

集群整体健康

| 关键指标           | 说明                                                   |
| ------------------ | ------------------------------------------------------ |
| 集群不可用节点比例 | 节点 Ready 状态占比                                    |
| 组件健康状态       | apiserver/etcd/scheduler/controller-manager 的运行状态 |
| 插件健康状态       | CoreDNS/CNI 插件是否正常运行                           |

- 资源对象状态

| 关键指标                   | 说明                                      |
| -------------------------- | ----------------------------------------- |
| 异常 Pod 数量              | Pending/Error/CrashLoopBackOff 状态的 Pod |
| 资源配额使用率（如果配置） | ResourceQuota 使用百分比                  |
| HPA 扩缩容活动             | 自动扩缩容的触发频率和当前副本数          |

- 集群事件

| 关键指标                   | 说明                                                 |
| -------------------------- | ---------------------------------------------------- |
| 关键事件实时监控           | 重点关注：Failed、BackOff、Evicted、Unhealthy 等事件 |
| 驱逐事件                   | 节点压力导致的 Pod 驱逐次数                          |
| OOM 事件                   | 容器 OOM Kill 的次数和位置                           |


- 证书管理

| 关键指标     | 说明                               |
| ------------ | ---------------------------------- |
| 证书过期时间 | kubelet 证书、etcd 证书、CA 证书等 |

## Master 组件

- API Server

| 关键指标             | 说明                             |
| -------------------- | -------------------------------- |
| 请求延迟（99 分位）    | 读写请求的处理耗时               |
| QPS（请求速率）      | 当前每秒处理的请求数             |
| `5xx`/`4xx` 错误率   | 请求失败的比例                   |
| 认证授权失败次数     | 认证失败、RBAC 权限拒绝次数      |
| 并发连接数           | 当前活跃连接数，是否接近限制     |

- Etcd

| 关键指标                     | 说明                                   |
| ---------------------------- | -------------------------------------- |
| 磁盘读写延迟（fsync 耗时）     | 直接影响集群响应速度                   |
| Raft 提案延迟                | 事务提交的延迟                         |
| Leader 选举次数              | 频繁选举表示集群不稳定                 |
| follower 与 leader 的同步延迟 | 数据同步的健康状况                     |
| DB 总大小                    | 是否接近配额（通常警告 >2GB，危险 >8GB） |
| 存储空间使用率               | etcd 数据目录的磁盘使用情况              |

- Scheduler

| 关键指标             | 说明                               |
| -------------------- | ---------------------------------- |
| 调度失败的 Pod 数量及原因 | 资源不足、端口冲突、污点不匹配等   |
| 调度算法执行耗时     | 从创建到绑定节点的耗时             |
| 待调度 Pod 数（Pending） | 长时间未调度的 Pod                 |

- Controller Manager

| 关键指标             | 说明                                     |
| -------------------- | ---------------------------------------- |
| 各个控制器的同步耗时 | Node/ReplicaSet/Deployment 等控制器的处理延迟 |
| 控制器重试次数       | 操作失败的次数                           |

## Node 组件

- 节点状态与压力

| 关键指标                      | 说明                           |
| ----------------------------- | ------------------------------ |
| Ready 状态                    | 节点是否可调度                 |
| 节点压力状态                  | DiskPressure/MemoryPressure/PIDPressure |
| 节点不可调度（Cordoned）      | 主动或被动标记为不可调度       |

- 资源利用率

| 关键指标         | 说明                               |
| ---------------- | ---------------------------------- |
| CPU 使用率       | 整体 CPU 使用百分比                |
| 内存使用率       | 整体内存使用百分比（不含缓存）     |
| 磁盘使用率       | 根分区/容器镜像存储分区的使用率    |
| inode 使用率     | 文件系统 inode 耗尽会导致无法创建文件 |
| 系统负载（Load Average） | 与 CPU 核心数对比评估系统压力      |

- 内核与网络

| 关键指标             | 说明                         |
| -------------------- | ---------------------------- |
| 文件描述符使用量     | 是否接近系统上限             |
| Conntrack 表使用率   | nf_conntrack_count / nf_conntrack_max |
| TCP 套接字错误       | 连接重置、超时等             |

## Pod 与容器（工作负载）

- 生命周期

| 关键指标             | 说明                                           |
| -------------------- | ---------------------------------------------- |
| Pod 阶段状态         | Pending/Running/Succeeded/Failed/Unknown       |
| Pod 启动时间         | 从创建到 Ready 的耗时                          |
| 容器重启次数         | 频繁重启需关注（CrashLoopBackOff）             |
| 镜像拉取失败次数     | ImagePullBackOff 状态，可能由于镜像不存在或认证失败 |

- 资源使用

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| CPU 使用量       | 实际使用的 CPU 核心数    |
| 内存使用量       | 实际使用的物理内存       |
| 网络流量         | 容器网络吞吐量           |
| 磁盘读写 IO      | 容器对磁盘的读写操作     |

- 资源饱和度与健康检查

| 关键指标             | 说明                           |
| -------------------- | ------------------------------ |
| CPU 节流时间         | CPU 使用超过 limit 被限制的时间  |
| OOM Kill 事件        | 容器因内存超限被杀死           |
| livenessProbe 状态   | 存活检查失败次数               |
| readinessProbe 状态  | 就绪检查失败次数               |

## 网络组件

- CoreDNS

| 关键指标                 | 说明                     |
| ------------------------ | ------------------------ |
| DNS 查询速率             | 每秒处理的 DNS 请求数    |
| DNS 查询延迟（99 分位）    | 从请求到响应的耗时       |
| DNS 错误率               | 超时、NXDOMAIN 等错误比例 |
| 与 kube-apiserver 通信错误 | 后端连接失败次数         |

- CNI 插件

| 关键指标             | 说明                     |
| -------------------- | ------------------------ |
| 网络策略数量         | 策略过多可能影响性能     |
| 策略执行延迟         | 网络策略生效的时间       |
| BGP 会话状态（Calico） | 路由协议连接状态         |
| BPF 映射大小（Cilium） | eBPF 内部数据结构大小    |

- Service

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| Endpoints 数量   | 后端 Pod 的数量是否正常  |
| 服务连通性       | 端到端的连通性检查       |

## 存储

- PV/PVC 状态

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| 卷状态           | Bound/Pending/Failed     |
| PVC 绑定延迟     | 从创建到绑定 PV 的耗时   |
| 存储配额使用率   | PVC 已用容量 / 总容量    |

- 存储性能

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| 存储 IO 延迟     | 读写操作的延迟           |
| 存储 IOPS        | 每秒读写次数             |
| 存储吞吐量       | 每秒读写字节数           |

- CSI 插件

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| CSI 调用延迟     | 存储插件接口的响应时间   |
| CSI 调用错误率   | 挂载/卸载/创建卷失败的次数 |

## 扩展组件

- Ingress Controller

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| 请求速率         | 每秒流入请求数           |
| 后端错误率（5xx） | 上游服务返回的 5xx 错误比例 |
| 响应延迟         | 请求处理的耗时           |
| TLS 握手耗时     | HTTPS 连接建立的延迟     |
| 活跃连接数       | 当前处理的并发连接数     |

- 外部负载均衡器

| 关键指标         | 说明                     |
| ---------------- | ------------------------ |
| 请求/错误率      | 请求量、错误率、请求健康状态 |
| 后端健康状态     | 检查的后端节点健康情况   |
