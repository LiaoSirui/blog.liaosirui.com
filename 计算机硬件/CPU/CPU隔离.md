CPU 隔离是一组强大的功能，可以为那些依赖特定且通常对延迟或性能有极端要求的工作负载设置提供支持

隔离 CPU4、5、6、7 核心

```bash
isolcpus=4,5,6,7

# isolcpus=4-7
```
