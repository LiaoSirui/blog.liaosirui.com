## Vector 使用

```yaml
sources:
  <unique_source_name>:
    # source configuration properties go here

transforms:
  <unique_transform_name>:
    # transform configuration properties go here

sinks:
  <unique_destination_name>:
    # sink configuration properties go here
```

### Sources

来源 input

### Transforms

变换 filter

### Sinks

接收器 output
