## token 命令

### 创建 token

创建新令牌

```bash
vault token create
```

指令 policy 创建

```bash
vault token create -policy=my-policy -policy=other-policy
```

创建一个周期性（periodic）令牌

```bash
ault token create -period=30m
```

### 吊销 token

吊销令牌及其子令牌

```bash
vault token revoke ...
```

吊销一个令牌但保留其子令牌（orphan模式）：

```bash
ault token revoke -mode=orphan ...
```

使用令牌访问器吊销令牌

```bash
vault token revoke -accessor ...
```

### 续约 token

```bash
vault token renew ...
```

续约特定时长

```bash
vault token renew -increment=30m ...
```

### 查看权限

```bash
vault token capabilities ... secret/creds
```
