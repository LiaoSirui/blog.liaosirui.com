## 连接 FTP 服务器

FTP 登录

```bash
ftp ftp.test.com
```

## 使用被动模式

登入 ftp 服务器之后默认使用的是主动模式，而有时候需要使用被动模式，输入`passive`命令即可进入被动模式。

## ftp 命令

键入`?`可以查看所有可用的 ftp 命令，`help command`可以查看命令的具体用法。

## 本地文件夹位置

使用 `lcd` 命令可以查看与改变本地文件夹位置

## 上传与下载文件

- `put a.txt` 可以将本地文件夹中的文件上传至服务器当前文件夹

- `get a.txt` 可以将服务器上的文件下载至本地

- `mput *` 可以上传多个文件

- `mget *` 可以下载多个文件
