S5700 支持 IPv6 协议栈，可配置 IPv6 地址并启用 ND（Neighbor Discovery）功能

启用 IPv6 功能

```bash
[HW-SW-01]ipv6
```

配置 IPv6 地址

```bash
[HW-SW-01]interface Vlanif 24
[HW-SW-01-Vlanif24]ipv6 enable
[HW-SW-01-Vlanif24]ipv6 address fd85:ee78:d8a6:8607::24:254/112
```

在双栈模式下，IPv4 和 IPv6 可同时运行并进行路由转发

```bash
# 启用 IPv6 路由功能
[HW-SW-01]ipv6 route-static fd85:ee78:d8a6:8607::24:0/112 2001:db8:1::2
 
# 配置 IPv4静态路由
[HW-SW-01]ip route-static 192.168.24.0 255.255.255.0 192.168.10.2
```
