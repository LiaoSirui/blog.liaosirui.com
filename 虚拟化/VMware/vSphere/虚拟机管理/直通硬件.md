## 直通板载 SATA 控制器

ESXi 默认禁止了 SATA 控制器直通，需 SSH 连接上 ESXi 服务器中，查找 SATA 控制器

```bash
lspci -v | grep "Class 0106" -B 1  
```

可以看到输出如下：其中 `8086：供应商 ID`；`8d62：设备 ID`；

```bash
0000:00:11.4 Mass storage controller SATA controller: Intel Corporation Wellsburg AHCI Controller
	 Class 0106: 8086:8d62
--
0000:00:1f.2 Mass storage controller SATA controller: Intel Corporation Wellsburg AHCI Controller
	 Class 0106: 8086:8d02
```

`passthru.map` 是 VMware ESXi 中一个可选的配置文件，用于定义哪些 PCI 设备可以被配置为 PCI 设备直通（PCI Passthrough），以便直接分配给虚拟机使用，编辑 `passthru.map`。

```bash
vi /etc/vmware/passthru.map
```

最后的行后添加：

```bash
# Intel Corporation Lynx Point AHCI Controller
# 供应商ID PCLE_ID
8086  8d62   d3d0    false
8086  8d02   d3d0    false
```

修改成功后，请点击【重新引导主机】，这时主机会进行重启。重启完毕后，可看到之前的灰色按钮变为了蓝色可点击状态，点击【切换直通】。切换直通后，可看到颜色变黄，还需【重新引导主机】，点击【重新引导主机】。
